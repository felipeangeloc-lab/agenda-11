🌊 Monitor de Reservatório
Simulador em Python que monitora o nível de água e exibe alertas coloridos no terminal conforme a criticidade.

🛠️ Requisitos
Python 3.x

Biblioteca colorama

Instalação:

Bash
pip install colorama
📊 Níveis e Cores
O sistema percorre os níveis de 1 a 5, atribuindo uma cor específica para cada status:

Nível 1: Vermelho (Crítico)

Nível 2: Amarelo (Baixo)

Nível 3: Verde (Médio)

Nível 4: Ciano (Alto)

Nível 5: Azul (Alerta Máximo)
